INSTRUCCIONES PARA SUBIR A GITHUB PAGES

1. Entra a tu repositorio de GitHub.
2. Sube estos archivos y carpetas:
   - index.html
   - style.css
   - script.js
   - assets/portada.png
3. En Settings > Pages, selecciona Deploy from a branch > main > /root.
4. Espera unos minutos y abre el enlace de GitHub Pages.

La página muestra solo la portada al inicio. El mensaje aparece al presionar "Abrir mi sorpresa".
El botón "Reproducir canción" abre YouTube con la canción indicada.
